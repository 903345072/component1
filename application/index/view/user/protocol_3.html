满盈策略服务协议

满盈策略服务协议（以下简称“本协议”）是平台运营方湖北满盈策略网络科技有限公司（以下简称“本公司”）与满盈策略用户（包含策略人和投资人，以下简称“用户”）就满盈策略服务相关事项订立的有效合约。本协议由协议正文以及本公司通过满盈策略已经发布的或将来可能发布的各类规则和文件的各项规定组成，如满盈策略发布的规则和文件与协议正文有冲突，以公布生效日期或签署日期在后的规则或文件为准执行。用户通过网络页面点击确认接受本协议，即表示用户与本公司已达成合意并同意接受本协议项下的全部约定内容。
在接受本协议之前，用户务必仔细阅读本协议的全部内容（特别是以粗体下划线标注的内容）。如果用户不同意本协议的任意内容，或者无法准确理解该条款的含义，请不要进行后续操作。

一、定义

本协议中，除非上下文另有解释，下列词语具有如下含义：
1、交易平台：指满盈策略互联网信息中介平台，由本公司运营。该平台及本公司根据业务需要不时修改的网址。
2、满盈策略：指满盈策略信息平台，面向满盈策略用户提供包括但不限于沪深所A股、50ETF、交易合作信息发布、交易合作撮合、交易指令通讯、交易风控管理、交易清结算（仅指数据结算）、资金支付（由支付机构托管）等服务的线上中介服务平台。
3、策略人：指通过满盈策略注册成为策略人的用户，作为投资人的交易合作方，负责向投资人提供交易策略的自然人。
4、投资人：投资人：指通过满盈策略注册成为投资人的用户，作为策略人的交易合作方，负责按策略人交易策略并利用自有资金和账户进行交易的自然人或法人或企业。
5、支付机构：指满盈策略委托的为满盈策略用户提供资金划转、查询、结算等支付服务的第三方支付机构。

二、满盈策略服务

满盈策略为用户提供以下服务：
1、交易合作撮合服务
满盈策略作为策略人和投资人的中间方，对策略人的交易谋略和投资人的交易需求进行居间介绍提供匹配参考，撮合双方建立一对一的交易合作关系。
2、交易指令通讯服务
满盈策略为策略人和投资人交易合作过程中提供向对方发送／接收交易指令通讯服务，交易指令包括：策略指令和点卖指令。满盈策略交易指令通讯速度从策略人发出交易指令被满盈策略确认时点至投资人接收到交易指令时点，两时点时间间隔原则不超过当天具体时间由投资人反馈。
3、交易风控管理服务
为最大程度确保策略人与投资人交易合作正常进行，保障双方利益，满盈策略将：  1）对存在退市、停牌等风险的交易品种，以风险警示形式（包括但不限于《策略风险警示股名单》、《递延风险警示股名单》等）实时发布  2）对于高风险交易品种要求策略人完成指定时长或指定数量的模拟交易后，方具备参与该交易品种交易合作资格。  3）根据策略人交易策略历史业绩，调整策略人参与交易合作的额度。  4）根据投资人与策略人交易数据和市场系统性风险分析，调整产品范围及具体方案等。
4、交易清结算服务
满盈策略根据策略人和投资人签署的《交易合作协议》相关约定，为双方提供交易清结算数据结算服务，数据结果对应金额资金结算由支付自制的支付机构完成，满盈策略明确告知策略人和投资人资金结算需取得相应资质，满盈策略不提供资金结算服务，满盈策略已委托具有结算资质的第三方支付机构提供服务。
5、资金支付服务
策略人与投资人交易合作过程发生的涉及交易款项的支付和结算都将由满盈策略委托连连第三方支付或其它有牌支付公司处理。双方应保证满盈策略账户状态正常，并确保账户中有足额的资金以便正常划转。因一方账户因满盈策略软件技术原因导致状态异常导致交易无法实现的，所有损失由满盈策略平台承担。
6、用户交易合作协议生成与保管
满盈策略为策略人和投资人生成交易合作协议供双方查询、存档。满盈策略会对策略人和投资人在满盈策略发生的交易信息及生成的协议保存，保存期限为【六】个月，如有疑问，策略人和投资人应及时向满盈策略提出书面查询单并附真实有效身份证明文件，超过保存期限要求查询的，由此产生的法律责任由策略人和投资人自行承担，与满盈策略人无关。
7、服务费及收费标准
策略人和投资人自愿同意向满盈策略支付一定的费用，具体收费事项及资费标准由满盈策略另行与用户约定或者通过满盈策略进行展示。
